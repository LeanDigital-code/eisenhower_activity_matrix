📊 The Eisenhower Matrix

The Eisenhower Matrix is a proven productivity method that helps you categorize tasks into four clear quadrants:

  • Do First → Urgent & Important
  • Schedule → Important but not urgent
  • Delegate → Urgent but not important
  • Eliminate → Not urgent & not important

This module brings this framework directly into Odoo Activities.
