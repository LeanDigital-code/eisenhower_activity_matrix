# Eisenhower Matrix

Modulo per **Odoo 18 Community Edition** per prioritizzare le attività del chatter con una matrice di Eisenhower.

## Funzionalità principali

- dashboard 2x2 Eisenhower
- trascinamento attività tra i quadranti
- priorità a stelle
- overview personale e di team
- viste list, kanban, pivot e graph

## Note tecniche

Questa versione è stata adattata per l'installazione su Odoo 18 CE, con correzioni di compatibilità lato manifest, viste e logica ORM per il cambio quadrante.
